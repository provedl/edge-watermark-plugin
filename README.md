# 网页水印助手 (Edge Watermark Plugin)

一个 Edge/Chrome 浏览器插件，可以在指定网站上显示全屏水印，支持为不同网站配置不同的水印文字。

## 功能

- 为不同网站设置独立的水印文字
- 全屏半透明水印，倾斜排列，不影响页面交互
- 支持子域名自动匹配
- 一键开关，随时启用/禁用
- 配置通过 Chrome Storage 同步，多设备共享

## 安装

1. 下载或克隆本仓库
2. 打开 Edge 浏览器，地址栏输入 `edge://extensions/`
3. 开启右上角「开发人员模式」
4. 点击「加载解压缩的扩展」，选择本项目文件夹

## 使用说明

### 添加水印规则

1. 点击浏览器工具栏中的插件图标，打开设置面板
2. 在「网站域名」输入框中填写目标网站（如 `example.com`）
3. 在「水印文字」输入框中填写该网站要显示的水印内容
4. 点击「添加」按钮

可以添加多条规则，每个网站显示各自独立的水印内容。

### 域名匹配规则

- 精确匹配：输入 `example.com`，仅匹配 `example.com`
- 子域名匹配：输入 `example.com`，同时匹配 `www.example.com`、`docs.example.com` 等所有子域名
- 无需输入 `http://` 或 `https://` 前缀，插件会自动处理

### 启用/禁用

- 设置面板顶部有「启用水印」开关，关闭后所有网站的水印都不会显示
- 重新打开即恢复，已配置的规则不会丢失

### 删除规则

- 在规则列表中，点击对应规则右侧的 `×` 按钮即可删除

### 多设备同步

插件配置通过 Chrome Storage Sync 存储，登录同一浏览器账号的设备会自动同步水印规则。

## 示例

| 网站 | 水印文字 |
|------|----------|
| company-a.com | 公司A内部文件 |
| company-b.com | 机密资料 禁止截图 |
| docs.internal.io | 仅供内部使用 |

## 注意事项

- 水印为纯前端覆盖层，不会修改网页内容
- 水印层设置了 `pointer-events: none`，不会影响页面正常点击和操作
- 水印 z-index 为最高值（2147483647），确保始终显示在最上层
- 如果网页使用了特殊的全屏模式，水印可能暂时不可见
